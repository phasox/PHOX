# ⚡ PHOX — Advanced Discord Bot by PhaseDev ⚡

---

## 🚀 Features

- ⚙️ Modular cog system (`/src`)
- 🌐 Per-server prefix system
- 💰 Economy & Casino
- 🔨 Moderation (mute, warn, nuke, etc.)
- 🎟️ Ticket system with transcripts
- ✅ Verification system
- 🧠 Word detection & auto role
- 📊 Logging system (ALL events)
- 🎯 AFK system (auto remove)
- 🧾 Snipe (last deleted messages)
- 🔐 Global blacklist system
- 📡 Rotating DND status

## ⚙️ Setup

### 1. Clone repo

```bash
git clone https://github.com/phasox/PHOX.git
cd PHOX
```

### 2. Install requirements

```bash
pip install -r requirements.txt
```

### 3. Add your token

Edit:

```python
config/token.py
```

```python
TOKEN = ""
```

---

### 4. Run the bot

```bash
python Main.py
```
